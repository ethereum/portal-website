# Why run a Portal Node?

The Portal Network gives you lightweight access to the Ethereum Protocol, directly from any consumer device. No more heavy lifting required by your machine.

## Trin-Desktop
The easiest way to get up & running is with Trin-Desktop.

Available for Windows, macOS, and Linux.

[Download the latest release](https://github.com/njgheorghita/trin-desktop/releases/tag/app-v0.1.0)

## Build from source
If you prefer to build the Portal Node from source for more fine-grained control over your node, you can follow any installation guides for a below Portal Client.

- [Trin](https://ethereum.github.io/trin/introduction/quickstart.html#quickstart)
- [Fluffy](https://fluffy.guide/quick-start.html)
- [Ultralight](https://github.com/ethereumjs/ultralight/)
